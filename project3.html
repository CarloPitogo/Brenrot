Project 3: Task Management Tool

Overview

This is a Trello-inspired task management tool designed to help individuals and teams organize tasks with visual boards and real-time updates.

Technologies Used

  • Vue.js (Frontend)
  • Firebase (Backend & Realtime Database)
  • Bootstrap (Styling)

Key Features

  • Kanban-style drag-and-drop task boards
  • Real-time collaboration with Firebase
  • Task comments, labels, and due dates
  • Authentication and persistent user sessions

Challenges Solved

It addressed the need for a simple, no-login-required workspace for quick team planning and idea management while supporting real-time syncing.
