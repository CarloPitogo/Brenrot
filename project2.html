Project 2: E-Commerce Web App

Overview

This e-commerce platform allows users to browse, search, and purchase products with a seamless checkout experience. It includes real-time cart updates and order tracking.

Technologies Used

  • React.js (Frontend)
  • Node.js + Express (Backend)
  • MongoDB (Database)
  • Stripe (Payments)

Key Features

  • User registration and login
  • Product listing with categories, filters, and ratings
  • Shopping cart and order history
  • Secure online payments with Stripe integration

Challenges Solved

The goal was to create a scalable, full-stack solution that mimics real-world e-commerce functionality with a focus on UI/UX and security.
